# 塔罗占卜网站

一个现代化的塔罗牌占卜网站，提供专业的塔罗牌解读服务。

## 功能特点

- 多种牌阵选择（单张、三张、凯尔特十字）
- 专业的塔罗牌解读
- 现代化的用户界面
- 流畅的动画效果

## 技术栈

### 后端
- Python Flask
- SQLAlchemy
- Flask-CORS
- Flask-RESTful

### 前端
- React
- Material-UI
- Framer Motion
- React Router

## 安装说明

### 后端设置

1. 进入后端目录：
```bash
cd backend
```

2. 创建虚拟环境：
```bash
python -m venv venv
```

3. 激活虚拟环境：
- Windows:
```bash
venv\Scripts\activate
```
- macOS/Linux:
```bash
source venv/bin/activate
```

4. 安装依赖：
```bash
pip install -r requirements.txt
```

5. 初始化数据库：
```bash
flask init-db
```

6. 运行后端服务器：
```bash
flask run
```

### 前端设置

1. 进入前端目录：
```bash
cd frontend
```

2. 安装依赖：
```bash
npm install
```

3. 运行开发服务器：
```bash
npm start
```

## 使用说明

1. 访问 http://localhost:3000 打开网站
2. 在占卜页面输入你的问题
3. 选择合适的牌阵
4. 点击"开始占卜"按钮
5. 查看解读结果
